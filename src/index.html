1. Introduction

តើអ្វីទៅជា Shell?

😂The shell is command interpreter ប្រើប្រាស់សម្រាប់បញ្ជា Linux Do Thing. The shell មិនគ្រាន់តែជា command interpreter ទេ ក៏ជា Programming Language ដែរ។

😃The Shell ចែកចេញជាពី Categories:

  • Grapical Shell : ប្រើ Interface
  • Command Line Shell : ប្រើ Command Line

😊ប្រភេទនៃ Shell នៅលើ Linux មានច្រើនប្រភេទ ដោយក្នុងនេះយើងនឹងរើសយក Bash Shell មកប្រើ

តើអ្វីទៅជា Bash Scripting?

😂Bash Scripting គឺជា Command line សរសេរក្នុង File ហើយអាចយកវាទៅ Run លើ Bash Program។ វាដំណើរការកូដ Line by Line។ Example : Create Folder, Navigation path… យើង អាច Save វានិង Run វាបាន ច្រើនដង។

តើសម្គាល់ Bash Script តាមរយៈអ្វី?

យើងអាចសម្គាស់ Bash Script តាមរយៈ ២ វិធី៖

  • File Extension of .sh
  • Scripts ចាប់ផ្តើមដោយ bash bang ឬ shebang #! bin/bash

របៀបបង្កើត File Bash Script

ក្នុងនេះដែរយើងបង្កើត File bash Script មួយឈ្មោះ touch hi_mom.sh

ស្វែងរក path របស់ Bash Shell ប្រើ Command which $SHELL

សរសេរ Command ក្នុង File

#! bin/bash

echo "Hi Mom!";

echo "I love Mom!";

echo "How are you doing? Mom!";

ដើម្បី Run Script ប្រើ Command : bash hi_mom.sh

លទ្ធផល

✍ យើងអាច Run File តាមរយៈ ១ វិធីទៀត គឺប្រើ ./hi_mom.sh
ប៉ុន្តែត្រូវ change user permission ហើយយើងអាច check មើល permission នៃ file ដោយប្រើ command ls -l hi_mom.sh
ដើម្បី change file permission ប្រើ command chmod +x hi_mom.sh

2. Variables

Variables គឺជាអញ្ញត្តិប្រើសម្រាប់ផ្ទុកនូវតំលៃ ឬទិន្នន័យ។ ឧទារហណ៍ សម្រាប់ផ្ទុកនូវឈ្មោះទំនិញ, តំលៃទំនិញ...។
ហើយទាក់ទឹងច្បាប់នៃការប្រកាសឈ្មោះនៃ Variable គឺដូចទៅនឹង Programming Language ផ្សេងៗទៀតដែរ។

Define and Access Variable

បន្ទាប់ពីប្រកាសឈ្មោះ Variable រួចយើងអាច ផ្តល់តម្លៃឲ្យបានដូចនឹង Programming ដ៏ទៃទៀតដែរគឺប្រើសញ្ញាស្មើរ។ ប្រសិនបើចង់ យកតម្លៃរបស់ Variable មកប្រើវិញ យើងត្រូវប្រើ Dollar Sign($)ដាក់ពីមុខឈ្មោះ Vairable។

Example:

#! bin/bash

Name="hi mom"; ផ្តល់តម្លៃទៅឲ្យអញ្ញាត្តិ

echo $Name;យកតម្លៃអញ្ញាត្តិមកប្រើវិញ

ប្រសិនបើយើងប្រកាស Variable ដែរមិនចង់ឲ្យគេអាចកែប្រែតំលៃបានគឺត្រូវប្រើ readonly variable_name

ប្រសិនបើយើងចង់ឲ្យ Variable មួយមិនអាច Ascess យកតម្លៃមកបានគឺត្រូវប្រើ unset variable_name

5. If Statement

If else statement ប្រើសម្រាប់កំណត់លក្ខខណ្ឌក្នុងការដំណើរកា shell ។ វាពិនិត្យមើលលក្ខខណ្ឌហើយដំណើកា statement 1 បើមិនពិតវាដំណើការ statement បន្ទាប់ ប្រសិនបើមិនលក្ខខណ្ឌពិតវានិងដំណើការ else statement ។

Example :

#! bin/bash

$a=2

if [ $a == 1 ]

echo "Value of a is one."

else if [ $a == 2 ]

echo "Value of a is two"

else

echo "None."

Output :

Value of a is one.
